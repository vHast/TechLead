Overview

Front-end web development is the development of the graphical user interface of a website, through the use of HTML, CSS, and Javascript, so that users can view and interact with that website.

These are the three pillars that are essential in this technology stack, after that you'll have to learn how to use libraries such as React or Bootstrap in order to enhance the user experience in your website.

The Path

You'll have to learn HTML5/CSS3 first, HTML5 is a markup language used for structuring and presenting content on the web, CSS3 stands for cascading style sheets, which is a style sheet language used for describing the look and formatting of a document written in a markup language.

To learn these two technologies first it works greatly to get the Responsive Web Design Certification, it will also teach you the basics of web design but you shouldn't worry about it since you're not a UI/UX designer, but it's always great to learn something new.

Make sure to finish the projects in the end, you should be able to create your own portfolio website or any tribute page of any personal interest you've got.

Responsive Web Design Certification

At this time you should get a GitHub and start getting around version control, I suggest using Git Bash if you're on Windows, get in and start uploading code, and more importantly, look into other people's code and try to understand it.

GitHub

Now we should be able to get our hands dirty with actual programming, we'll have to learn Javascript for that, which is a programming language that is one of the core technologies of the web, over 97% of the websites use Javascript on the client side for web page behavior

It will be useful if one day you'll want to go to learn Backend development and get in touch with Node.js

In order to start learning Javascript you should complete the following certification.

Javascript Algorithms and Data Structures

I must warn you that this course is not very intuitive, it misses some important aspects to Web Development such as the DOM (Document Object Model) and more complex algorithm solving, so I'm going to list a few resources that will help you to achieve that knowledge.

Javascript.info Eloquent Javascript: DOM CodeWars: Algorithm Problems

This is where a huge bulk of time should be invested in, the next step will be learning a Javascript framework, but I must encourage you to learn the foundations of Javascript since it will be very useful when getting ahead of any JS framework.

And at this point I would encourage you to start collaborating in any HTML/CSS/JS project in GitHub, and start working into your own profile page to include all the projects you've done so far.

The next step should be learning React, which is included in the Front End Development Libraries certification in freeCodeCamp
